# About Me

I'm **Labhansh Vashisht**, an undergraduate Computer Science student specializing in **Cyber Security** at **Jaypee Institute of Information Technology (JIIT), Noida**, while simultaneously pursuing a **B.Sc. in Data Science & Artificial Intelligence** from **IIT Guwahati**.

My primary interests lie in designing scalable software systems, solving algorithmic problems, and developing practical applications that bridge software engineering with artificial intelligence. I enjoy transforming ideas into production-ready solutions while continually exploring modern technologies across the software stack.

Over the past year I have worked with **Python, JavaScript, TypeScript, C, C++, R**, modern web technologies, machine learning libraries, cloud deployment platforms, and developer tooling. Alongside software development, I actively study cybersecurity fundamentals, secure software engineering practices, and open-source development.

I believe engineering is more than writing code—it is about building reliable, maintainable, and impactful products that solve real-world problems.
